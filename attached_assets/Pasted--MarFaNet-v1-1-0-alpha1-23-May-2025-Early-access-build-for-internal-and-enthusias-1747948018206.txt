## MarFaNet **v1.1.0‑alpha1** – *23 May 2025*

> **Early‑access build** for internal and enthusiast testers. Expect rough edges; please file feedback in GitHub → Discussions → “alpha1 feedback”.

---

### 🚀 Highlights

| Category             | Feature                        | Notes                                                                                             |
| -------------------- | ------------------------------ | ------------------------------------------------------------------------------------------------- |
| **Networking**       | **WireGuard Engine (Opt‑In)**  | Toggle in **Settings → Engine**. Benchmarked **⚡ 31 % faster** throughput vs Xray on 5 G.         |
| **Privacy/Routing**  | **Split‑Tunnelling UI**        | Per‑app allow/deny list; persists to Room DB; rules injected into runtime Xray/WG config.         |
| **UX / Monitoring**  | **Real‑Time Stats Overlay**    | Floating overlay shows live RTT & up/down speeds; dismissible. Enable via **Settings → Overlay**. |
| **UI Modernisation** | **Jetpack Compose HomeScreen** | Material 3 “Dynamic Color” support; dark/light & pure‑black AMOLED themes.                        |
| **Platform Upgrade** | AGP 9.0 + Kotlin 2.0 (K2)      | Faster build times, future‑proof for Android 15.                                                  |

---

### 🛠 Known Issues

1. **Overlay permission** may need manual enable on MIUI/EMUI.
2. WireGuard tunnel fails on some Android 9 devices – falls back to Xray automatically.
3. Split‑Tunnelling does not yet support system apps.

### 📥 Installation

1. Download **MarFaNet‑1.1.0‑alpha1‑universal.apk** from this release.
2. Verify SHA‑256 against above.
3. On device: *Settings → Security → Install unknown apps* → allow for browser/file‑manager.
4. Tap APK, install over existing MarFaNet (profiles retained).
